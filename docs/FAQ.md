## 1. 서비스 소개

### 번개가입은 어떤 서비스인가요?

번개가입은 카페24 쇼핑몰에 소셜 로그인을 추가하는 SSO 앱입니다.

구글, 카카오, 네이버, 애플, 디스코드, 텔레그램 6종의 소셜 프로바이더를 지원하며(LINE, X, Facebook 2026년 지원 예정), 1클릭 회원가입으로 쇼핑몰의 가입 전환율을 높일 수 있습니다. 복잡한 회원가입 양식 없이 기존 소셜 계정으로 즉시 가입이 가능합니다.

| 기능 | Free | Plus (월 6,900원) |

| 가입 쿠폰 | 1종 (정률 할인) | 3종 (무료배송/정액/정률) |
| "powered by 번개가입" 제거 | ❌ | ✅ |

### 무료로 사용할 수 있나요?

네, Free 플랜은 무제한 무료로 사용할 수 있습니다.

| 플랜 | 금액 | 주요 기능 |
|------|------|-----------|
| **Free** | 무료 | 소셜 로그인 6종+ + 가입 쿠폰 1종 |
| **Plus** | 월 6,900원 / 연 79,000원 | 미니배너, 이탈 감지 팝업, 에스컬레이션, AI 보고서, 쿠폰 3종 등 |

### 카페24에서 기본 제공하는 소셜 로그인과 뭐가 다른가요?

카페24가 기본 제공하는 소셜 로그인은 4종(구글, 카카오, 네이버, 페이스북)에 불과하지만, 번개가입은 현재 6종(구글, 카카오, 네이버, 애플, 디스코드, 텔레그램)을 지원하며 LINE, X, Facebook (2026년 지원 예정)을 추가할 예정입니다.

또한 번개가입은 단순 소셜 로그인 연결 이상의 마케팅 기능을 함께 제공합니다:

- AI 마케팅 카피 자동 생성
- 미니배너(가입 유도 배너)
- 이탈 감지 팝업
- 신규 가입 시 쿠폰 자동 발급
- 에스컬레이션(재방문 감지 혜택 제공)
- AI 주간 보고서

## 2. 설치 및 설정

### 설치 후 가장 먼저 해야 할 것은?

**SSO 설정 가이드**에 따라 카페24 관리자에서 SSO 연동을 등록해야 합니다. 이 설정이 완료되어야 쇼핑몰 로그인 페이지에 소셜 로그인 버튼이 표시됩니다.

순서:
1. 대시보드 > 설정 > **SSO 설정 가이드** 접속
2. 가이드에 표시된 값(Client ID, Secret, URL 등)을 카페24 관리자에 입력
3. 카페24 관리자 > 쇼핑몰 설정 > 고객 설정 > **SSO 로그인 연동 관리**에서 연동 등록

→ [SSO 설정 가이드 바로가기](/dashboard/settings/sso-guide)

### "회원가입 시 SNS 계정 연동" 설정은 왜 활성화해야 하나요?

이 설정은 기존 ID/PW 회원이 소셜 로그인을 시도할 때 동일 이메일로 기존 계정과 자동 연동되도록 하는 카페24 옵션입니다.

**이 설정을 활성화하지 않으면** 소셜 로그인 시 기존 회원과 별도로 `@s` 접미사가 붙은 새 회원이 생성됩니다. 이 경우 주문 내역, 적립금, 등급 등이 기존 계정과 완전히 분리됩니다.

활성화 위치: 카페24 관리자 > 쇼핑몰 설정 > 고객 설정 > 회원 설정 > **회원가입 시 SNS 계정 연동 사용함**

### SSO 설정에서 입력해야 할 값들은 어디서 확인하나요?

대시보드 > 설정 > **SSO 설정 가이드** 페이지에서 아래 값들을 모두 복사할 수 있습니다:

- Client ID
- Client Secret
- Authorize Redirect URL
- Access Token Return API
- User info Return API

→ [SSO 설정 가이드 바로가기](/dashboard/settings/sso-guide)

### SSO 설정이 정상인지 어떻게 확인하나요?

카페24 관리자에서 SSO 등록을 완료한 후, 대시보드 > 설정 > **SSO 설정 가이드** 페이지 하단의 **'설정 확인'** 버튼을 클릭하세요.

번개가입이 카페24의 SSO 슬롯(sso~sso5)을 자동으로 스캔합니다:
- **번개가입** (초록): 정상 등록됨
- **다른 앱** (노랑): 다른 SSO 앱이 등록됨
- **미등록** (회색): SSO 미사용 슬롯

번개가입 슬롯이 감지되면 자동으로 설정이 확정됩니다.

## 3. 소셜 로그인 동작

### 소셜 로그인하면 새 회원이 만들어지나요?

"회원가입 시 SNS 계정 연동" 설정 여부에 따라 다릅니다:

- **활성화 시**: 동일 이메일을 가진 기존 회원이 있으면 연동 화면이 표시됩니다. 기존 회원을 선택하면 계정이 연동되고 새 회원이 생성되지 않습니다.
- **미활성화 시**: 항상 새 `@s` 회원이 생성됩니다.

### 프로바이더별 주의사항이 있나요?

| 프로바이더 | 이메일 제공 | 기존 회원 연동 |
|------------|------------|----------------|
| 구글, 카카오, 네이버 | 제공 | 안정적 |
| 애플 | 조건부 제공 | Hide My Email 사용 시 연동 불가할 수 있음 |
| X(트위터), 텔레그램 | 미제공 | 기존 회원 연동 불가 |

- **Apple**: "Hide My Email" 기능으로 임시 이메일을 사용하는 경우 기존 계정과 이메일 매칭이 불가할 수 있습니다.
- **X(트위터), 텔레그램**: 이메일을 제공하지 않으므로 기존 ID/PW 회원과의 연동이 불가합니다. 항상 새 `@s` 회원이 생성됩니다.

### 로그인 페이지에 소셜 버튼이 안 보여요

다음을 확인하세요:

1. SSO 연동 설정이 "**사용함**"으로 설정되어 있는지 확인
2. 카페24 관리자에서 SSO 등록이 완료되었는지 확인
3. **중요**: 회원가입 페이지(`/member/join.html`)에서는 SSO 버튼이 표시되지 않습니다. 이는 카페24의 플랫폼 제한 사항으로, **로그인 페이지에서만** 소셜 로그인 버튼이 동작합니다.

## 4. 다른 SSO 앱에서 전환

### 다른 SSO 앱에서 번개가입으로 전환하면 기존 회원은 어떻게 되나요?

기존 회원의 데이터(주문, 적립금, 등급 등)는 카페24에 유지됩니다. 단, SSO 앱이 변경되면 기존 소셜 로그인 연동이 끊어집니다.

- "회원가입 시 SNS 계정 연동" 활성화가 **필수**입니다.
- 동일 이메일로 번개가입을 통해 소셜 로그인하면 재연동이 가능합니다.
- 단, 이전 앱에서 생성된 `@s` 회원이 있다면 중복 계정이 생길 수 있습니다.

### 전환 시 주의사항은?

1. 기존 SSO 앱을 삭제하기 **전에** 카페24 관리자에서 "SNS 계정 연동" 설정을 먼저 활성화합니다.
2. 기존 앱에서 생성된 `@s` 회원이 있으면 중복 계정 발생에 주의합니다.
3. 전환 후 일정 기간(1~2주) 동안 기존 회원의 로그인 관련 문의에 대응할 준비를 하세요.

## 5. 회원 관리 주의사항

### 소셜 회원(@s)을 탈퇴시키면 어떻게 되나요?

**주의**: 소셜 회원(`@s` 계정)을 탈퇴 처리하면, 동일 이메일로 연동된 **모든 SSO 로그인이 차단**됩니다. 카페24가 이메일 기준으로 탈퇴 이력을 관리하기 때문입니다.

권장 대안:
1. "회원가입 시 SNS 계정 연동" 설정으로 ID/PW 회원에 소셜 계정을 연동합니다.
2. 연동된 상태에서 SNS 연동만 해제 처리합니다.
3. 이렇게 하면 소셜 로그인 차단 없이 `@s` 계정만 정리할 수 있습니다.

## 6. 쿠폰

### 가입 쿠폰이 자동으로 발급되나요?

네, 대시보드 기본 설정에서 쿠폰을 설정하고 저장하면 카페24에 쿠폰이 자동 생성됩니다. 이후 신규 회원가입 시 자동으로 쿠폰이 발급됩니다.

| 플랜 | 쿠폰 종류 |
|------|-----------|
| Free | 1종 (정률 할인) |
| Plus | 3종 (무료배송, 정액 할인, 정률 할인) |

→ [기본 설정 바로가기](/dashboard/settings/general)

### 쿠폰이 사용되었는지 확인할 수 있나요?

현재 카페24 API의 제한으로 번개가입에서 쿠폰 사용 상태를 직접 추적하는 것이 불가합니다. 쿠폰 사용 여부는 카페24 관리자에서 직접 확인해야 합니다.

- **발급 히스토리**: 대시보드 > 쿠폰 현황에서 확인 가능
- **사용 현황**: 카페24 관리자 > 프로모션 > 쿠폰 관리에서 확인

→ [쿠폰 현황 바로가기](/dashboard/settings/coupon)

## 7. Plus 기능

### Plus에서 무엇이 추가되나요?

Free 플랜의 모든 기능에 더해 다음이 제공됩니다:

| 기능 | 설명 |
|------|------|
| **미니배너** | 쇼핑몰 내 소셜 가입 유도 배너 |
| **이탈 감지 팝업** | 페이지 이탈 시 소셜 가입 유도 팝업 |
| **에스컬레이션** | 재방문 감지 후 자동 혜택 제공 |
| **카카오 채널 연동** | 신규 가입 시 카카오 채널 추가 유도 |
| **AI 주간 보고서** | 가입 성과 주간 분석 리포트 |
| **AI 추천 마케팅 문구** | 쇼핑몰 맞춤 AI 카피 자동 생성 |

→ [Plus 업그레이드](/dashboard/billing)

### Free에서 Plus로 업그레이드하면 기존 설정이 유지되나요?

네, 모든 기존 설정은 그대로 유지되며 Plus 기능이 추가로 활성화됩니다.

다운그레이드 시에는 Plus 기능이 즉시 비활성화되지만, 설정 데이터는 보존됩니다. 재구독 시 이전 설정이 그대로 복원됩니다.

### 카카오톡 채널 자동 연결은 어떻게 쓰나요?

Plus 요금제에서 제공되는 기능으로, 고객이 카카오 로그인 후 쇼핑몰의 카카오톡 채널로 자동 친구 추가를 유도합니다. 관리자 > 설정 > 카카오 채널 ID 입력 후 활성화하시면 동작합니다.

## 8. 앱 삭제

### 앱을 삭제하면 어떻게 되나요?

- 쇼핑몰 로그인 페이지에서 소셜 로그인 버튼이 사라집니다.
- 기존 소셜 가입 회원 데이터는 카페24에 유지됩니다.
- 소셜 로그인으로 재접속이 불가해집니다. (단, 기존 ID/PW 회원에 소셜 계정이 연동된 경우 ID/PW 로그인은 계속 가능합니다.)
- 재설치하면 이전 설정이 복원됩니다.

## 9. AI 기능

### AI 정체성 분석이란 무엇인가요?

쇼핑몰 URL의 HTML과 판매 상품 정보를 AI가 분석하여 **업종**(패션/뷰티/식품 등), **타겟 고객**(20-30대 여성 등), **톤앤매너**(친근하고 캐주얼 등), **핵심 키워드**를 자동으로 파악합니다.

기본 설정 페이지에서 실행하며, 일일 10회까지 가능합니다. 이 정보가 AI 보고서와 추천 문구의 정확도를 높이는 기반이 됩니다.

### AI 주간 보고서는 어떻게 동작하나요?

매주 **월요일 오전 9시(KST)**에 자동 생성됩니다. 보고서에는 4가지 핵심 내용이 포함됩니다:

1. **지난주 성과 요약** — 데이터 기반 사실 분석
2. **이번 주 전략 제안** — 번개가입 기능 범위 내 실행 가능한 전략
3. **실행 가능한 액션 3가지** — 바로 적용 가능한 구체적 행동
4. **AI 인사이트** — 앱 범위 밖의 참고 트렌드

이전 보고서와 비교하여 변화 추이도 분석합니다. AI 보고서 페이지에서 수동으로도 생성할 수 있습니다.

### AI 추천 마케팅 문구란 무엇인가요?

AI 보고서 생성 시 쇼핑몰 정체성에 맞는 **마케팅 문구 7종**이 자동 생성됩니다:
- 미니배너 문구
- 에스컬레이션 토스트 메시지
- 에스컬레이션 플로팅 배너 문구 + 버튼 텍스트
- 이탈 감지 팝업 제목 + 본문 + CTA 버튼

각 설정 페이지에서 'AI 추천' 영역에 추천 문구가 표시되며, 적용 버튼으로 원클릭 적용할 수 있습니다.

### AI 추천 문구 자동 적용이란?

기본 설정에서 'AI 추천 문구 자동 적용' 토글을 켜면, AI 보고서가 생성될 때마다 추천 문구가 미니배너, 이탈 감지 팝업, 에스컬레이션 설정에 자동 반영됩니다.

매주 월요일 자동 보고서와 함께 마케팅 문구도 자동 업데이트되므로, 항상 최신 트렌드에 맞는 문구가 유지됩니다.

### AI 기능을 사용하려면 무엇이 필요한가요?

AI 기능은 **Plus 플랜 전용**입니다. 먼저 기본 설정에서 **AI 정체성 분석**을 실행해야 보고서와 추천 문구의 품질이 높아집니다.

정체성 분석 없이도 동작하지만, 쇼핑몰 맞춤화된 결과를 원하시면 정체성 분석을 먼저 진행하세요.

### AI 기능별 일일 호출 한도는?

번개가입의 AI 기능은 남용 방지를 위해 일별/월별 호출 제한이 있습니다.

- AI 쇼핑몰 정체성 분석: **일 10회** (Free/Plus 공통)
- AI 주간 브리핑: **일 5회** (Plus 전용)
- AI 에스컬레이션 메시지 생성: **일 5회** (Plus 전용)
- AI 추천 마케팅 문구: **월 10회** (Plus 전용)

한도에 도달하면 다음날(또는 다음 달) 자정(KST)에 리셋됩니다.
