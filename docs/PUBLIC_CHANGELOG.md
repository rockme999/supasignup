## 2026년 5월 (v2.5.1) — 위젯 마케팅 강화 + UX 정돈 🎁

로그인 위젯이 단순한 가입 입구를 넘어 **마케팅 도구**가 됩니다. 쿠폰팩과 안내 문구를 위젯 안에 직접 노출해 가입 전환율을 끌어올리세요.

### 🎁 쿠폰팩이 위젯 안으로 — 가입 전 미리보기 (Plus)

- 가입 시 받는 **5장 ₩55,000 가치 쿠폰팩**이 이제 *이탈 팝업뿐 아니라 로그인 위젯 자체*에 카드로 표시됩니다
- 노출 위치(소셜 위/아래), 간격(0~60px), 크기(LG/MD/SM/XS), 디자인 4종(다크/브랜드/일러스트/미니멀), 반짝 효과를 자유롭게 조정
- 이탈 팝업의 쿠폰팩 카드와 **디자인·애니메이션이 자동 동기화** — 한 곳에서 바꾸면 양쪽 같이 바뀝니다

### 📝 위젯 안내 텍스트 2종 (Plus)

- **텍스트1** (상단 타이틀 아래, 작은 글씨) — 가입/로그인 편의성 강조 (기본 ON, "아이디 비밀번호 입력없이 번개가입! 번개로그인!")
- **텍스트2** (소셜 버튼과 쿠폰팩 사이, 큰 볼드) — 쿠폰팩 증정 등 임팩트 문구 (기본 ON, "회원가입 즉시 사용가능한 쿠폰팩 증정")
- 다크 프리셋(글래스모피즘·네온 글로우·리퀴드 글래스)에서도 가독성이 자동 보정됩니다
- AI 추천 문구도 이 2개 영역에 자동 적용됩니다 (월요일 자동 브리핑과 함께 갱신)

### ↩️ 가입 / 로그인 후 원래 페이지로 자동 복귀

- 상품 상세 페이지 → 로그인 페이지 → 소셜 가입 → **원래 보던 상품 페이지로 자동 이동**
- 가입 직전 보던 상품/카테고리/장바구니 컨텍스트를 잃지 않아 가입 직후 이탈 방지

### 🎨 로그인 디자인 페이지 — Accordion 재구성

- 위젯 디자인 카드를 **3개 섹션 accordion**으로 재구성 — "기본 선택 / 세부 조정 / 추가 옵션"
- 각 섹션마다 [기본값][저장] 버튼 + 카드 외부에 [전체 기본값][전체 저장] 버튼
- 모든 섹션은 기본 접힘 — 필요할 때만 열어 보세요
- 상단 타이틀이 **기본 ON**으로 변경 (이전: OFF)

### ✨ 위젯 미리보기 ↔ 실제 위젯 시각 통일

- 어드민 미리보기와 실제 쇼핑몰 위젯의 아이콘/폰트/path가 완전히 동일하게 렌더링됩니다
- 다크 프리셋의 Google 아이콘만 4색 G 로고 유지, 다른 프로바이더는 흰 fill로 자동 보정

### 🤖 AI 추천 문구 영역 확장

- AI가 추천하는 마케팅 문구가 7종 → **9종**으로 확장 (위젯 안내 텍스트1·2 추가)
- AI가 *현재 활성 기능*을 정확히 알도록 컨텍스트 강화 — 쿠폰팩이 활성이면 "5장 ₩55,000 쿠폰팩 증정" 같은 정확한 카피, 비활성이면 임의 카피 생성 안 함

### 🔧 카카오 채널 — 기본 설정 페이지로 통합

- 카카오 채널 ID 설정이 별도 사이드 메뉴 항목에서 제거되고 **기본 설정 페이지 최상단**으로 이동
- 한 번에 모든 기본 설정을 관리 — 페이지 이동 없이 SSO/혜택/카카오 채널을 한 화면에서 처리
- 기존 URL(`/dashboard/settings/kakao`)은 자동 리다이렉트되므로 기존 즐겨찾기는 그대로 동작

### 🎫 가입 쿠폰 정책 단순화 — Free에서도 3종 모두

- 무료 플랜에서도 **무료배송 / 정액 할인 / 정률 할인 3종 모두 동시 활성화** 가능 (이전: 1종만)
- 금액·할인율·기간·최소주문 모두 **자유 설정** (Plus와 동일)
- Plus의 차별점은 **웰컴 쿠폰팩 5장(₩55,000 가치)** 단계별 자동 발급 — 쿠폰 정책이 더 명확해졌습니다

### ✨ Plus 승급 + 쿠폰팩 활성화 시 위젯 자동 노출

- Free에서 Plus로 승급하고 쿠폰팩을 활성화하면 **위젯의 쿠폰팩 카드와 안내 텍스트1·2가 자동으로 켜집니다**
- 운영자가 별도로 토글을 만지지 않아도 가입 마케팅이 즉시 작동

### 🛡 안정성 / 호환성 개선 (내부)

- 카페24 쿠폰 자동 발급 토글 API의 2026-03-01 버전 호환 (status pause/restart 패턴 + 멱등성 가드)
- Plus → Free 다운그레이드 시 쿠폰팩 자동 정지(5장 일괄), Plus 복귀 시 자동 재개 (어드민/웹훅/구독 만료/스케줄 cron 모든 경로 일관 처리)
- 일부 브라우저(Comet, Brave Shields 등 내장 추적 보호)에서 위젯 통계 호출이 차단되던 문제 해결 — 통계 정확도 5~25% 개선
- 새로고침 시 의도치 않게 자동 redirect되던 부작용 차단 (3분 TTL 가드)

## 2026년 5월 (v2.5.0) — 로그인 디자인 대폭 개선 ✨

로그인 디자인 페이지가 한층 더 강력해졌습니다. 프로바이더가 늘어도 화면이 길어지지 않고, 쿠폰팩 미리보기도 한 곳에서 확인할 수 있어요.

### ✨ 아이콘 모드 프로바이더 — 컴팩트하게, 모두를 위한 가입 경로

- 자주 쓰이는 카카오·네이버는 **풀버튼**으로, 디스코드·텔레그램 같은 보조 프로바이더는 **하단 작은 아이콘**으로 운영자가 직접 선택할 수 있어요
- 선택 부담은 줄이고, 모든 프로바이더의 가입 경로는 그대로 유지됩니다
- 직전에 로그인한 프로바이더는 모드와 무관하게 풀버튼으로 자동 promote ("이전에 사용" 라벨)
- Plus 프리셋(글래스모피즘·네온 등)에서도 정상 동작 — 다크 배경에서는 아이콘이 자동으로 흰색 처리됩니다

### 🎁 로그인 디자인 페이지 — Plus 쿠폰팩 미리보기 추가

- 로그인 디자인 페이지에서 가입 시 발급되는 **웰컴 쿠폰팩 카드 미리보기**를 직접 확인할 수 있어요
- 4단계 크기(LG/MD/SM/XS) 토글로 쇼핑몰 페이지에서 어떻게 보일지 즉시 비교 가능

### 🔧 카카오 채널 — Free 플랜에서도 사용 가능

- 카카오 채널 ID 설정이 **Plus 전용에서 Free로 이동**했습니다 — 이제 모든 운영자가 사용 가능
- 별도 메뉴에서 **기본 설정** 페이지 안으로 통합 — 한 곳에서 모든 기본 설정을 관리할 수 있어요

### 🛠 그 외 다듬기

- 미니배너 & 라이브카운터 통합 페이지 + 미니배너 활성화 토글 + 페이지 숨김 옵션
- 이탈 감지 팝업 페이지 4-카드 재구성 + 쿠폰팩 카드 통합 (Exit-intent 단독 페이지 제거)
- 쿠폰팩 카페24 등록 422 오류 수정 + 5장 정상 등록 동작
- Plus 프리셋 6종 모바일 호버 시연 패턴 통일
- 위젯 PC/모바일 폰트·아이콘 path를 어드민 미리보기와 완전 동기화

## 2026년 4월 (v2.0.0) — 카페24 공식 앱 정식 출시 🎉

번개가입이 카페24 공식 앱스토어에 정식 등록되었습니다.
더 많은 기능, 더 합리적인 가격으로 새롭게 선보입니다.

### 💎 Plus 플랜 — 가격이 달라졌습니다

- 무료 플랜이 이제 **회원 수 제한 없이 현재 무료 제공 중**입니다 (이전에는 월 100명 제한이 있었어요)
- Plus 플랜 **월 6,900원** — 매달 부담 없이 시작
- **연 79,000원** — 결제 공백 없이 1년 안심 운영. 비슷한 가입 유도 앱들의 *월 비용* 수준의 연간 가격
- 부담 없는 가격으로 모든 강력한 기능을 누리세요

### ✨ Plus 전용 — 가입 유도 도구 3종 세트

쇼핑몰을 방문하는 비회원을 회원으로 바꿔주는 도구입니다.

- **미니 배너** — 로그인·가입 페이지 상단과 하단에 "지금 가입하면 X원 할인!" 같은 혜택 문구를 띄워드려요. 슬라이드 카드 최대 4종까지 설정 가능합니다
- **이탈 감지 팝업** — 페이지를 떠나려는 비회원에게 마지막으로 가입 혜택을 한 번 더 보여드려요 (PC에서는 마우스가 창 밖으로 나갈 때, 모바일에서는 위로 스크롤할 때 자동 감지)
- **재방문 에스컬레이션** — 자주 오는데 아직 가입하지 않은 비회원에게 방문 횟수에 따라 혜택 강도를 자동으로 조절해드려요 (1~2회 방문 → 부드럽게, 5회 이상 → 좀 더 적극적으로)

### ✨ Plus 전용 — AI 4가지 기능

- **쇼핑몰 정체성 분석** — AI가 쇼핑몰의 분위기, 주 고객층, 특징을 자동으로 파악해 마케팅 카피 작성에 활용합니다
- **주간 AI 브리핑** — 매주 가입 현황과 인사이트를 AI가 요약해서 보내드려요
- **맞춤 카피 생성** — 쇼핑몰 특성에 맞는 가입 유도 문구를 AI가 자동 생성합니다
- **재방문 맞춤 메시지** — 재방문 비회원에게 맞춤형 문구를 자동으로 노출합니다

### 📊 Plus 전용 — 상세 전환율 분석

- 배너 노출 → 클릭 → 가입 완료 → 쿠폰 발급까지 단계별 전환율을 확인할 수 있습니다
- "어떤 도구가 가입에 실제로 기여하는지" 한눈에 파악하세요

### ✨ 무료 포함 — 모두를 위한 신기능

- **AI 문의 자동 답변** — 1:1 문의에 AI가 초안을 작성해드려요. 운영자가 직접 답변하기 전에 AI 초안을 검토·수정해서 발송할 수 있습니다
- **1:1 문의 게시판** — 이제 대시보드 안에서 바로 문의하고 답변을 받을 수 있어요. 이미지 첨부도 지원합니다
- **사용 가이드·퀵스타트·FAQ** — 처음 설치할 때 헷갈리지 않도록 자세한 가이드를 제공합니다
- **카카오 채널 자동 친구 추가** — 가입 완료 시 쇼핑몰 카카오 비즈채널을 자동으로 안내합니다 (Plus)
- **"Powered by 번개가입" 브랜딩 제거** — 쇼핑몰 고객에게 번개가입 로고가 보이지 않도록 설정할 수 있습니다 (Plus)

### 🚀 대시보드 개선

- 전체 UI를 화이트·심플 스타일로 새롭게 디자인했습니다
- 위젯 디자인 커스터마이징이 크게 강화되었습니다
  - 5가지 스타일 프리셋 (컬러·모노톤·테두리·호버채움·아이콘만)
  - 크기·간격·둥글기·정렬 등 6가지 슬라이더로 세밀하게 조정
  - 9가지 소셜 아이콘과 안내 문구를 직접 커스텀
- 통계 페이지에서 더 많은 데이터를 더 빠르게 확인할 수 있습니다

### 🔒 보안 강화

이번 버전에서 여러 보안 개선이 이루어졌습니다. 사용자 개인정보 보호와 시스템 안전성을 높였습니다.
