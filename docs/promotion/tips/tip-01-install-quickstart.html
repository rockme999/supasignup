이 가이드로 얻는 것

5분 뒤, 우리 쇼핑몰 로그인 페이지에 ● 카카오 버튼이 뜹니다.

고객이 그 버튼을 누르면 비밀번호 입력·본인인증 없이 1초 만에 가입이 끝납니다. 진짜로.

준비물

  • 카페24 운영자 계정 하나만 있으면 됩니다
  • 번개가입은 무료라 결제 정보 등록 필요 없음
  • 카카오/네이버/구글 개발자 계정? 필요 없습니다. 번개가입이 대신 처리합니다
⏱ 전체 소요 시간 — 약 5분 (설치 1분 · 설정 3분 · 확인 1분)

1 카페24 앱스토어에서 설치 · 1분

  1. 카페24 앱스토어 검색창에 "번개가입" 입력
  2. 검색 결과에서 번개가입 ⚡ 선택 후 [설치] 클릭

💡 체크 — 앱 상세 페이지에 "무료" 표시가 있는지 확인하세요. 결제 정보 요구가 없어야 정상입니다.

2 권한 동의 후 대시보드 진입 · 1분

설치 과정에서 아래 권한 동의 화면이 나타납니다. 모두 번개가입이 정상 동작하는 데 필요한 최소 권한입니다.

어떤 권한이 왜 필요한가
  • 회원 읽기/쓰기 — 소셜 로그인으로 들어온 고객을 쇼핑몰 회원으로 생성
  • 스크립트 태그 쓰기 — 로그인 버튼 위젯을 쇼핑몰에 자동 삽입
  • 주문 읽기 — 재방문·재구매 분석을 위한 읽기 전용(쓰기 없음)

동의를 마치면 아래와 같은 대시보드 홈이 열립니다.

3 프로바이더 켜기 + 순서 조정 · 2분

좌측 메뉴에서 로그인 디자인(또는 소셜 프로바이더 설정)으로 이동합니다. 기본은 Google / Kakao / Naver 3종이 켜져 있으니, 필요에 따라 Apple / Discord / Telegram도 토글하세요.

👍 한국 쇼핑몰 기준 추천 순서

카카오 → 네이버 → 구글 → Apple 순서가 가입 전환율이 가장 높습니다. Discord / Telegram은 커뮤니티·게임·디지털 상품 쇼핑몰에서만 노출을 권장합니다.

순서는 프로바이더 행 오른쪽 ▲ ▼ 버튼으로 조정합니다. 변경 즉시 미리보기에 반영됩니다.

4 실제 쇼핑몰에서 확인 · 1분

관리자 우측 상단의 [저장] 버튼을 누르고, 쇼핑몰의 로그인 페이지를 새 탭에서 열어 확인합니다. 시크릿 창에서 열면 로그인 상태 방해 없이 깔끔하게 볼 수 있습니다.

✅ 성공 기준
  • 로그인 페이지에 카카오(또는 선택한) 버튼이 보인다
  • 버튼 클릭 시 카카오 로그인 팝업이 뜬다
  • 로그인 승인 후 쇼핑몰 마이페이지로 이동하며, 회원이 새로 생성된다

자주 겪는 상황 3가지

Q1. 버튼이 로그인 페이지에 안 보여요

쇼핑몰 스킨 구조에 따라 버튼 삽입 위치가 잡히지 않을 수 있습니다. 로그인 디자인 > 위젯 삽입 위치에서 "로그인 폼 위" 또는 "커스텀 셀렉터"로 바꿔보세요. 카페24 공용 스킨은 대부분 .login__button, .login__sns에서 잘 동작합니다.

Q2. 기존 이메일 회원과 소셜 회원은 분리되나요?

같은 이메일이면 기존 회원과 자동으로 연결됩니다. 카페24 회원 DB는 그대로, 소셜 로그인은 추가 인증 수단으로만 붙습니다.

Q3. 앱을 삭제하면 쇼핑몰에 남은 버튼은요?

앱 삭제 시 번개가입이 삽입한 위젯 스크립트도 함께 제거됩니다. 기존 이메일 로그인 폼은 그대로 유지됩니다.

  • 카카오 로그인 원리 — 카카오 개발자 앱 등록 없이 어떻게 가능한지 궁금하다면 "번개가입이 카카오 로그인을 자동 연동하는 방법" 사용팁을 참고하세요
  • 통계 확인 — 대시보드 홈에서 오늘/이번달/누적 가입 숫자와 프로바이더별 비율을 실시간 확인
