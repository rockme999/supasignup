# 사용팁 #1 — 번개가입 설치 퀵스타트

> 카페24 파트너센터 > 프로모션 > 사용팁 > 등록

## 입력 폼 값

| 필드 | 값 | Bytes |
|---|---|---|
| 스토어 | 한국 (KR) | — |
| 제목 `contents_name` | `번개가입 설치하고 5분 안에 소셜 가입 오픈하기` | 64 / 80 |
| 요약 `contents_desc` | `앱 설치부터 첫 카카오 1클릭 가입까지 5분 안에 끝내는 단계별 가이드.` | 94 / 120 |
| 대표이미지 | `../images/main/tip-01-main.png` (444×320) |
| 노출 | 심사 승인 후 ON |
| 관련 상품 | **번개가입** |
| 상세 내용 HTML | `./tip-01-install-quickstart.html` |

## 본문 구성 요약

**심사 기준 주의**: 상품상세와 같은 기능 나열 금지 → 대신 **실제 운영자가 따라할 수 있는 단계별 절차**와 **각 단계의 기대 결과**를 명확히 보여줄 것.

1. **이 가이드로 얻는 것** — 5분 후 상태 정의 (고객이 카카오 버튼 누르면 가입 완료)
2. **준비물** — 카페24 운영자 계정만 있으면 OK (무료 앱이라 결제 정보 X)
3. **STEP 1 (1분)** — 앱스토어 설치
4. **STEP 2 (1분)** — 권한 동의 & 첫 대시보드
5. **STEP 3 (2분)** — 프로바이더 On/Off + 순서 변경
6. **STEP 4 (1분)** — 쇼핑몰에서 실제로 보이는지 확인
7. **자주 겪는 상황 3가지** (FAQ성)
8. **다음 단계** — 디자인 커스텀/카카오 연동 사용팁 링크
