## 본문 구성 요약

핵심 메시지: **"소셜 로그인은 Free로 충분하지만, 가입을 실제 매출로 바꾸려면 Plus 기능이 필요하다. 그게 월 6,900원."**

1. **인트로** — 소셜 로그인을 깔았는데도 매출이 안 오르는 이유: 가입 자체를 안 한다 / 가입했는데 재방문 안 한다 / 운영자가 카피·배너 만들 시간이 없다
2. **Plus가 해결하는 3가지 문제** 소개 (가입 유도 / 재방문 / 운영 자동화)
3. **기능 묶음 #1 — 가입 유도 위젯 3종**: 미니 배너 / 이탈 감지 팝업 / 재방문 비회원 에스컬레이션
4. **기능 묶음 #2 — AI 4종**: 쇼핑몰 정체성 자동 설정 / 주간 AI 브리핑 / 맥락 기반 카피 생성 / 재방문 맞춤 메시지
5. **기능 묶음 #3 — 운영 보너스**: 카카오 채널 자동 친구 추가 / 가입 전환 퍼널 통계 / "Powered by" 브랜딩 제거
6. **가격 강조** — 월 6,900원이 어떤 가치인가 (외주 디자인 1건/카피라이터 1시간/광고비 5분 vs Plus 한 달)
7. **연간 결제 시 절감액** — 연 79,000원 = 월 환산 약 6,583원 (약 5% 절감)
8. **CTA** — Plus 업그레이드 버튼 / 무료부터 써보고 결정

## 심사 포인트

- **유익성**: 운영자가 Plus를 어떤 문제 해결에 쓰는지 구체 시나리오로 제시 (가입 유도 → 재방문 → 매출 전환)
- **중복성**: 스토리 #1(런칭 소개), #2(디자인 슬라이더)와 Plus라는 주제로 명확히 구분됨 — #1은 개요, #3은 Plus 심층
- **가격 비교 표현**: 타사 명칭 직접 명시 ❌, "외주 디자인 1건 / 카피라이터 1시간" 같은 일반 가치 비교만 사용
- **AI 결과의 단정 표현 회피**: "매출이 X% 오릅니다" 같은 효과 보장 표현 ❌. "가입 전환에 도움" 정도 중립 톤
- **카카오/네이버/구글 로고**: 본문 직접 삽입 ❌. 모두 자체 렌더링한 일러스트만 사용
- **이미지 alt**: 본문 HTML 내 모든 `<img>` 태그에 alt 포함 ✅
- **미구현 기능 표기**: 본문에서 "곧 출시" 표현 회피하고, 현재 Plus 패키지 설명으로 일관 (서비스계획서 v2 기준 Plus에 포함된 기능 일체)
