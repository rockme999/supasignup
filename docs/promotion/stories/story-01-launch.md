# 스토리 #1 — 번개가입 런칭 소개

> 카페24 파트너센터 > 프로모션 > 스토리 > 등록

## 입력 폼 값

| 필드 | 값 | Bytes |
|---|---|---|
| 스토어 선택 | 스토어 / 한국 (KR) | — |
| 제목 `contents_name` | `번개 가입이 매출을 바꾼다 — 카페24 공식 앱 런칭 ⚡` | 71 / 80 |
| 요약 `contents_desc` | `가입 폼에서 떠나는 고객을 카카오·네이버·구글·애플 1클릭으로 붙잡는 카페24 공식 앱.` | 118 / 120 |
| 대표이미지 타입 | 이미지 (또는 YouTube URL 있으면 동영상) |
| 대표이미지 파일 | `../images/main/story-01-main.png` (444×320, 500KB 이하) |
| 노출 | 심사 승인 후 ON |
| 관련 상품 | **번개가입** (앱 선택) |
| 상세 내용 HTML | `./story-01-launch.html` 내용을 에디터 코드 보기에 붙여넣기 |

## 본문 구성 요약

1. **문제 제기** — 긴 회원가입 폼이 구매 전환율을 갉아먹는다
2. **솔루션** — 소셜 6종 1클릭 가입, 설치 1분
3. **관리자 대시보드** — 가입 현황/프로바이더 분포 한눈에
4. **디자인 커스텀** — 슬라이더로 즉시
5. **Plus AI 마케팅** — 쿠폰·팝업 자동 설계
6. **가격 비교** — 타사 대비 10배 이상 저렴
7. **CTA** — 앱스토어에서 설치

## 심사 포인트

- **유익성**: "가입 폼 이탈 → 매출 하락" 이라는 운영자 체감 문제를 명확히 제기하고 해결책 제시 ✅
- **중복성**: 기존 카페24 스토리와 유사 주제 있는지 검색 필요 (소셜 로그인 앱은 있어도 "슬라이더로 디자인" 차별점으로 차별화)
- **가격 비교 근거**: 타사 서비스 이름을 직접 명시하지 않고 "경쟁 A" 등 익명 처리 + 공개된 가격표 기준이라고 명시
- **이미지 alt**: 본문 HTML 내 모든 `<img>` 태그에 alt 포함 ✅
