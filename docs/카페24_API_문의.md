> ⚠️ **중요 전제 조건 — 연동 기능의 동작 범위:**
>
> 이미 해당 소셜 계정으로 SSO를 통해 @s 회원이 생성되어 있다면, 동일 이메일이더라도 연동 화면이 뜨지 않는다. 카페24가 "신규 가입"이 아닌 "기존 @s 회원 로그인"으로 처리하기 때문이다.
>
> 즉, 연동 기능은 **해당 소셜 계정으로 처음 SSO 가입할 때만** 동작한다.
>
> 따라서 운영 시:
> - 쇼핑몰에 번개가입 앱 설치 시 "회원가입 시 SNS 계정 연동" 설정을 **먼저** 활성화해야 함
> - 이미 @s 회원이 생성된 후에는 연동 불가 — 해당 @s 회원을 탈퇴 처리하더라도 카페24에서 재가입이 거부됨

> ⚠️ **위험 — 소셜 계정 탈퇴 시 동일 이메일의 모든 SSO 로그인 차단:**
>
> 카페24는 이메일 기준으로 탈퇴 이력을 관리한다. 동일 이메일을 사용하는 소셜 계정(@s) 중 하나가 탈퇴되면, 해당 이메일을 사용하는 다른 소셜 계정으로의 SSO 로그인 및 신규 가입도 "탈퇴한 계정"으로 차단된다.
>
> **예시:**
> - aaa@aaa.com 이메일의 ID/PW 계정(aaa)과 구글 @s 계정이 있음 (연동되지 않은 상태)
> - 구글 @s 계정을 회원 탈퇴
> - 이후 aaa@aaa.com 이메일의 카카오, 네이버 등 다른 소셜로 SSO 로그인/가입 시도 → "탈퇴한 계정" 메시지로 차단됨
> - ID/PW 계정(aaa)은 별도로 존재하므로 영향 없음
>
> **운영 시 주의사항:**
> - 쇼핑몰 운영자에게 소셜 회원(@s) 탈퇴 시 해당 이메일의 모든 SSO 로그인이 차단됨을 안내해야 함
