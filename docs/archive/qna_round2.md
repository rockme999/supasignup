# SupaSignup Q&A Round 2 - Claude의 분석 및 추가 질문

## Round 1 답변 정리 (확인된 사항)

### 핵심 철학
- **"거의 회원가입 절차 없이"** 쇼핑몰 이용 가능하게 하는 것이 최우선 목적
- 사용자는 이미 구글/카카오/네이버 등에 로그인된 상태일 확률이 높음
- OAuth에서 아무 개인정보가 안 넘어와도, 유니크 ID만 있으면 임시 회원 생성
- 상세 정보는 주문 시점에 수집 (Progressive Profiling)

### 비즈니스 모델
- 프리미엄: 무료 100명 → 유료 월 29,900원 / 연 329,900원 (추후 변경 가능)
- 경쟁사 "1초가입" 대비 가격 경쟁력 있음 (1초가입: 월 49,000~69,900원 + 부가세)

### UI 방향
- 팝업 모달 X → 쇼핑몰 회원가입 페이지에 버튼으로 삽입
- 이미 로그인된 OAuth면 클릭 한 번으로 바로 가입

### 데이터 처리
- 쇼핑몰 플랫폼 API로 직접 회원 생성이 가능하면 그렇게 처리
- 불가능하거나 부족하면 우리 DB에 임시 저장 후 나중에 동기화
- **각 플랫폼의 회원 정보 쓰기 API 조사 필요** (이건 제가 조사하겠습니다)  
--> 이미 다른 서비스에서 어느 정도 조사가 된게 있으니 참고로 하도록 특히 veripack-multi프로젝트에 가장 많이 조사되어 있다.

### 기술 스택
- Supabase 사용 안 함 (확정)
- DB 필요 시: 로컬 서버 + Cloudflare Tunnel
- DB 불필요 시: Cloudflare Workers로 처리
- RDBMS 필요하면 서비스 활성화 전까지는 로컬 운영

## Vanilla JS SDK 설명

제가 언급한 "임베디드 위젯용 Vanilla JS SDK"를 설명드리면:

쇼핑몰 운영자가 회원가입 페이지에 아래와 같은 코드를 붙여넣는 방식입니다:

```html
<!-- SupaSignup 버튼 삽입 코드 -->
<script src="https://cdn.supasignup.com/sdk.js"></script>
<script>
  SupaSignup.init({
    shopId: 'my-shop-123',        // 쇼핑몰 고유 ID
    platform: 'cafe24',           // 플랫폼 종류
    providers: ['google', 'kakao', 'naver', 'apple'],  // 사용할 OAuth
    target: '#signup-buttons'     // 버튼이 들어갈 위치
  });
</script>
```

이렇게 하면 해당 위치에 "구글로 가입", "카카오로 가입" 등의 버튼이 자동 생성됩니다.
"Vanilla JS"라는 건 React/Vue 같은 프레임워크 없이 순수 JavaScript로 만든다는 뜻이고, "SDK"는 쇼핑몰에 삽입하는 라이브러리를 말합니다.
--> 쇼핑몰에서는 단지 위에 스크립트를 복사해서 넣는 것 뿐이지? OK!


**이 방식이 "간단하게 코드를 삽입"하는 요구사항에 가장 적합합니다.**

다만, 카페24 같은 경우는 앱스토어를 통해 설치하는 방식이 더 자연스러울 수 있어서 플랫폼별로 연동 방식이 달라질 수 있습니다.
--> 앱스토어를 통해서 어떻게 로그인화면을 바꿀 수 있지???

### Q2. 우선 지원 플랫폼

--> 일단은 카페24인데 이미 멀티 플랫폼으로 진행한 다른 프로젝트들이 있어서 구현이 어려울거 같지는 않다. 우선은 카페24로

--> 좋아 일단 4개로 하고 서비스 오픈전에 디스코드, 텔레그램, 페이스북, 인스타그램, 틱톡? 등을 추가하자.
--> 우선은 oauth가 가능한 소셜서비스가 뭐가 있고 한국에서 얼마나 상시 사용하고 있는지 조사가 필요하다. 

### Q4. 경쟁사 "1초가입" 대비 차별점
- "1초가입"은 카카오 개인정보 동의를 통한 자동 연동, 등급/적립금/쿠폰 발급 등 부가기능 제공
- 우리의 핵심 차별점은 무엇으로 가져갈 건지?
  - (a) 가격 경쟁력 (49,000원 vs 29,900원)
  - (b) 멀티 플랫폼 지원 (1초가입은 카페24만)
  - (c) 더 많은 OAuth 지원 (Discord, Telegram 등)
  - (d) 더 간단한 설치/사용 경험
  - (e) 기타?
- 추가 서비스(등급, 적립금, 쿠폰 등)를 포함할지 여부도 여기서 결정하면 좋겠습니다

--> 더 많은 outh를 지원하고 더 간단한 설치와 가입을 강조하자. 

### Q5. 카페24 앱스토어 등록
