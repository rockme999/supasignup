# 번개가입 Q&A Round 5 - SSO 조사 결과 + OAuth 조사 결과

## 1. 카페24 SSO 심층 조사 결과 (매우 중요)

### 핵심 발견: 카페24 SSO에서 "우리가 OAuth Provider"가 되어야 한다

카페24 SSO 연동의 구조:
```
카페24 = OAuth Consumer (Client)
번개가입 = OAuth Provider (Authorization Server)  ← 우리가 만들어야 함
```

카페24 SSO 설정에 입력하는 항목:
| 항목 | 설명 | 번개가입에서 제공 |
|------|------|-----------------|
| Client ID | 번개가입이 발급 | bg_shop_xxxxx |
| Client Secret | 번개가입이 발급 | 자동 생성 |
| Authorize Redirect URL | 번개가입 인증 페이지 | `https://bg.suparain.kr/oauth/authorize` |
| Access Token Return API | 번개가입 토큰 발급 | `https://bg.suparain.kr/oauth/token` |
| User Info Return API | 번개가입 사용자 정보 | `https://bg.suparain.kr/oauth/userinfo` |

### SSO 동작 플로우

```
1. 사용자가 카페24 쇼핑몰 로그인 페이지에서 "번개가입으로 로그인" 클릭
2. 카페24가 번개가입 Authorize URL로 리다이렉트
3. 번개가입 인증 페이지에서 "구글/카카오/네이버/애플로 가입" 버튼 표시
4. 사용자가 원하는 OAuth 선택 (예: 구글)
5. 구글 인증 완료 → 번개가입 서버에 사용자 정보 저장
6. 번개가입이 Authorization Code를 카페24로 반환
7. 카페24가 번개가입 Token API로 Access Token 요청
8. 카페24가 번개가입 UserInfo API로 사용자 정보 요청
9. 카페24가 자동으로 회원 생성 + 로그인 세션 생성 ✅
```

### 이 방식의 장점
- **카페24가 회원 생성과 로그인 세션을 모두 자동 처리** → 우리 부담 최소
- 카페24 공식 지원 기능 → 안정적
- SSO 가입 회원은 아이디에 "@s" 접미사 → 일반 회원과 구분 가능
- 최초 1회만 약관 동의 팝업 (사전 동의 시 생략 가능)

### 이 방식의 단점/제약

1. **SSO 설정을 API로 자동화 불가** → 쇼핑몰 운영자가 카페24 관리자에서 수동 설정 필요
2. **SSO 연동 최대 5개** → 한 쇼핑몰에 5개까지만 등록 가능 (우리는 1개면 충분)
3. **우리가 OAuth Provider를 구축해야 함** → 서버 필요 (DB도 필요)
4. **로그인 버튼이 카페24 로그인 페이지에 표시됨** → 회원가입 페이지가 아닌 로그인 페이지

### SSO 설정 수동 가이드 (운영자에게 안내 필요)
1. 카페24 관리자 > 쇼핑몰 설정 > 고객 설정 > SSO 로그인 연동 관리
2. "신규 연동 등록" 클릭
3. "사용 여부"를 "사용함"으로 변경
4. 번개가입에서 발급받은 Client ID, Client Secret 입력
5. 번개가입이 안내하는 3개의 URL 입력
6. 저장

→ 이 과정을 가이드 문서 + 영상으로 제공하면 될 것 같습니다.

## 2. 경쟁사 "1초가입" 추정 기술 분석

"1초가입"의 API 권한에 `mall.write_customer`(회원 쓰기)가 없음에도 회원가입이 되는 이유:

**추정 방식 2가지:**

**(A) 카페24 SSO 연동 활용** (유력)
- 킵그로우 서버가 OAuth Provider 역할
- 카카오 인증 후 킵그로우 서버에서 사용자 정보 처리
- 카페24 SSO를 통해 자동 회원 생성/로그인
- "@k" 접미사가 붙는 이유: SSO를 통해 카카오로 가입한 회원

**(B) ScriptTag + 프론트엔드 폼 제출**
- ScriptTag로 삽입한 JS가 카페24 회원가입 폼을 프로그래밍 방식으로 제출
- 비공식적이지만 기술적으로 가능

**시사점**: 1초가입도 결국 SSO 또는 프론트엔드 해킹 방식을 사용할 수밖에 없음. 우리도 같은 선택지.
