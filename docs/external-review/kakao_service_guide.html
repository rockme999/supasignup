⚡ 번개가입 (BungaeGaib)

쇼핑몰 소셜 로그인 솔루션 — 서비스 소개 및 이용 동선

1 서비스 개요

번개가입은 카페24 쇼핑몰에 소셜 로그인(카카오, 네이버, 구글 등)을 1-클릭으로 연동하는 SaaS 서비스입니다.
쇼핑몰 고객은 SNS 계정으로 간편하게 회원가입/로그인하고, 쇼핑몰 운영자는 관리자 대시보드에서 모든 설정을 통합 관리합니다.

🔐

소셜 로그인 연동

카카오, 네이버, 구글, 애플, 토스, 디스코드, 텔레그램, 틱톡 등 9종 프로바이더 지원

📊

관리자 대시보드

소셜별 가입 통계, 쇼핑몰별 현황, 플랜/과금 관리를 한 곳에서 확인

💳

카페24 인앱결제 연동

카페24 앱스토어 인앱결제 기반의 구독 플랜 자동 관리 및 정기 과금

2 서비스 이용 동선

쇼핑몰 고객이 카카오 로그인을 선택한 후 회원 가입 완료까지의 전체 흐름입니다.

1
Step 1
쇼핑몰 고객이 로그인 페이지 방문
쇼핑몰 고객은 번개가입이 연동된 카페24 쇼핑몰의 로그인/회원가입 페이지에 접속합니다.
번개가입이 제공하는 소셜 로그인 버튼 UI가 표시되며, 고객은 원하는 SNS 계정을 선택합니다.

Step 2
카카오 로그인 동의 화면
고객이 "카카오로 시작하기"를 클릭하면 카카오 OAuth 인증 화면으로 이동합니다.
카카오가 제공하는 공식 동의 화면에서 번개가입(쇼핑몰)이 요청하는 정보 제공에 동의합니다.

Step 3
로그인 완료 — 쇼핑몰 회원 자동 가입
카카오 인증이 완료되면 번개가입 서버가 카카오 프로필 정보를 받아 카페24 쇼핑몰 회원으로 자동 가입 처리합니다.
신규 고객은 회원가입과 로그인이 동시에 완료되며, 기존 회원은 카카오 계정이 연결됩니다.
