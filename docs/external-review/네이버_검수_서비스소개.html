1 서비스 개요
번개가입이 무엇인지, 누구를 위한 서비스인지 설명합니다.

서비스 한 줄 정의

번개가입은 카페24 쇼핑몰에 설치하는 소셜 로그인 기반 회원가입 솔루션입니다.
쇼핑몰 방문 고객이 네이버·카카오·구글·애플 계정을 이용해 1-클릭으로 쇼핑몰 회원가입 및 로그인을 완료할 수 있도록 합니다.
복잡한 가입 양식 입력 없이 소셜 계정 인증만으로 즉시 쇼핑몰 회원이 됩니다.

대상 사용자 B2B2C

  • 쇼핑몰 운영자 번개가입 대시보드를 통해 서비스를 설치·설정하는 카페24 쇼핑몰 운영 사업자
  • 쇼핑몰 고객 해당 쇼핑몰을 방문하여 소셜 계정으로 회원가입하는 일반 소비자

핵심 가치

  • 가입 간소화 소셜 계정 1회 인증으로 쇼핑몰 회원가입 즉시 완료
  • 플랫폼 연동 카페24 SSO를 통해 소셜 인증 결과를 쇼핑몰 계정으로 자동 변환
  • 운영 편의 관리자 대시보드에서 가입 통계·위젯 디자인·프로바이더 설정 관리

서비스 URL

  • 서비스 메인 https://bg.suparain.kr
  • 개인정보처리방침 https://bg.suparain.kr/privacy

위젯 로드 — 쇼핑몰 회원가입 페이지에 번개가입 위젯이 표시됨

카페24 쇼핑몰 운영자가 번개가입 스크립트를 설치하면, 회원가입/로그인 페이지에 소셜 로그인 버튼 위젯이 자동으로 삽입됩니다. 고객은 "네이버로 시작하기" 버튼을 확인합니다.

쇼핑몰 고객 번개가입 위젯

네이버 로그인 버튼 클릭 — OAuth 2.0 인증 플로우 시작

고객이 "네이버로 시작하기" 버튼을 클릭하면, 번개가입 서버가 생성한 네이버 OAuth 인증 URL로 리다이렉트됩니다. 네이버 로그인 화면이 표시됩니다.

쇼핑몰 고객 네이버 로그인 번개가입 서버

네이버 계정 인증 — 동의 및 인증 완료

고객이 네이버 계정으로 로그인하고 정보 제공에 동의합니다. 네이버 인증 서버는 authorization code를 번개가입 서버의 콜백 URL로 전달합니다.

쇼핑몰 고객 네이버 로그인

사용자 정보 수신 — 번개가입 서버에서 토큰 교환 및 프로필 조회

번개가입 서버는 authorization code를 access token으로 교환하고, 네이버 프로필 API를 호출하여 이메일과 이름을 수신합니다. 수신된 정보는 카페24 회원 생성에만 사용됩니다.

번개가입 서버 네이버 로그인

카페24 SSO 회원가입 — 소셜 정보로 쇼핑몰 회원 자동 생성

번개가입 서버는 카페24 SSO API를 통해 수신된 이메일·이름으로 쇼핑몰 회원 계정을 자동 생성합니다. 기존 회원이면 로그인 처리됩니다.

번개가입 서버 카페24 SSO

회원가입 완료 — 고객이 쇼핑몰에 로그인된 상태로 진입

카페24가 발급한 로그인 세션과 함께 고객은 쇼핑몰 메인으로 리다이렉트됩니다. 별도의 추가 입력 없이 회원가입과 로그인이 동시에 완료됩니다.

3 주요 기능 설명
번개가입의 세 가지 핵심 기능 구성 요소를 설명합니다.
WIDGET
소셜 로그인 위젯
쇼핑몰 회원가입·로그인 페이지에 삽입되는 버튼 UI 컴포넌트

번개가입 위젯은 카페24 쇼핑몰의 회원가입 및 로그인 페이지에 JavaScript 스니펫 한 줄로 삽입됩니다. 위젯은 운영자가 활성화한 소셜 프로바이더(네이버, 카카오, 구글, 애플)의 버튼을 자동으로 표시합니다.

  • 지원 프로바이더: 네이버, 카카오, 구글, 애플 (운영자가 개별 활성화)
  • 쇼핑몰 테마 색상에 맞춰 버튼 색상·크기·레이아웃 커스터마이징 가능
  • 모바일·PC 반응형 디자인 지원
  • 스크립트 삽입만으로 설치 완료 (별도 서버 설치 불필요)
SSO
카페24 SSO 연동
소셜 인증과 카페24 회원 시스템을 연결하는 인증 중개 서비스

번개가입의 핵심 기술은 소셜 OAuth 인증 결과를 카페24 SSO(Single Sign-On) API로 전달하여 쇼핑몰 회원 계정을 자동으로 생성·연결하는 인증 중개 역할입니다.

  • 소셜 OAuth 2.0 인증 → 번개가입 서버 → 카페24 SSO API 순서로 인증 체인 구성
  • 네이버에서 수신한 이메일·이름으로 카페24 쇼핑몰 회원 자동 생성
  • 기존 쇼핑몰 회원이면 소셜 계정과 연결 후 자동 로그인 처리
  • 카페24 멀티쇼핑몰 구조 지원 (쇼핑몰별 개별 설정)
DASHBOARD
관리자 대시보드
쇼핑몰 운영자가 번개가입 서비스를 설정·운영하는 관리 도구

쇼핑몰 운영자는 번개가입 대시보드(https://bg.suparain.kr)에서 서비스 전반을 설정하고 가입 현황을 모니터링합니다.

  • 가입 통계: 소셜 프로바이더별·기간별 회원가입 수 현황
  • 프로바이더 설정: 사용할 소셜 로그인 항목 활성화/비활성화
  • 위젯 커스터마이징: 버튼 색상, 크기, 텍스트, 표시 순서 등 조정
  • 설치 가이드: 카페24 스크립트 삽입 방법 안내 및 코드 복사
4 네이버 로그인 활용 방식
번개가입에서 네이버 로그인 API가 어떤 목적으로, 어떤 방식으로 사용되는지 상세히 설명합니다.

네이버 로그인 사용 목적

번개가입은 카페24 쇼핑몰 방문 고객이 네이버 계정을 이용해 쇼핑몰 회원가입 및 로그인을 1-클릭으로 완료할 수 있도록 하기 위해 네이버 로그인 API를 사용합니다.

네이버 로그인은 쇼핑몰 자체 회원가입 양식(아이디/비밀번호/이메일 입력 등)을 대체하는 인증 수단으로만 활용됩니다. 소셜 인증이 완료되면 수신된 정보(이메일, 이름)는 카페24 쇼핑몰 회원 계정 생성에만 사용되며, 다른 목적으로 활용되지 않습니다.

사용 주체

  • 서비스명 번개가입 (BungaeGaib)
  • 인증 방식 OAuth 2.0 Authorization Code Flow
  • 콜백 URL https://bg.suparain.kr/auth/callback/naver

사용 맥락

  • 표시 위치 카페24 쇼핑몰 회원가입 및 로그인 페이지의 번개가입 위젯 내
  • 버튼 텍스트 "네이버로 시작하기" 또는 운영자 설정 텍스트
  • 인증 후 처리 네이버 프로필 조회 → 카페24 SSO 회원 생성 → 쇼핑몰 로그인

네이버 로그인이 사용되지 않는 용도

  • 마케팅 수신된 사용자 정보를 마케팅·광고 목적으로 사용하지 않음
  • 제3자 제공 네이버에서 수신한 정보를 제3자에게 제공하지 않음 (카페24로의 SSO 연동 제외)
  • 프로필 연동 네이버 게시물, 친구 목록, 쇼핑 내역 등 소셜 활동 정보에 접근하지 않음

6 개인정보 처리
수집된 개인정보의 보호 방법과 처리 방침을 설명합니다.

암호화 저장

네이버에서 수신한 개인정보(이메일, 이름, 고유 식별자)는 번개가입 서버 데이터베이스에 AES-256-GCM 방식으로 암호화하여 저장됩니다. 평문으로 저장되지 않습니다.

AES-256-GCM 암호화

전송 구간 보호

모든 데이터 송수신은 HTTPS(TLS 1.2 이상)를 통해 이루어집니다. 네이버 인증 서버와 번개가입 서버, 카페24 API 간의 모든 통신이 암호화됩니다.

HTTPS / TLS 1.2+

보존 기간 및 파기

개인정보는 서비스 이용 기간 동안 보존됩니다. 이용자가 연동 해제 또는 회원 탈퇴를 요청하면 관련 개인정보를 즉시 삭제 처리합니다.

탈퇴 요청 시 즉시 삭제

제3자 제공 제한

수집된 개인정보는 카페24 SSO 회원가입 처리에 필요한 최소한의 범위에서만 카페24 플랫폼으로 전달됩니다. 다른 제3자에게 제공되지 않습니다.

카페24 SSO 처리 목적만

접근 통제

개인정보에 접근 가능한 직원을 최소화하고, 접근 이력을 기록·관리합니다. 불필요한 개인정보 접근이 발생하지 않도록 내부 접근 권한을 엄격히 제한합니다.
