## 5. 민감정보 가드레일 체크리스트

프롬프트 조립 시 최종 검수:
- [ ] `schema.sql` 의 테이블명/컬럼명이 프롬프트에 없는가
- [ ] `database_id`, `KV namespace id`, `account_id` 같은 식별자가 없는가
- [ ] OAuth client_id/secret 부분 문자열이 없는가
- [ ] 내부 API 엔드포인트(`/api/admin/...` 등)가 운영자 노출 범위 밖의 경로는 빠졌는가
- [ ] "결정 필요 사항"/"TODO"/"A안 vs B안" 같은 의사결정 과정이 유출되지 않는가
