## 4. 번개가입 SWOT 분석

### Strengths (강점)
- **9종 소셜 프로바이더** — 경쟁사 대비 압도적 다양성 (경쟁사는 1~4종)
- **해외 소셜 지원** — Google, Apple, Discord, LINE, X, Telegram 등 글로벌 대응
- **ScriptTag 기반** — 쇼핑몰 전 페이지에 스크립트 삽입 → 배너/팝업 등 확장 가능
- **기존 회원 연동** — 카페24 기본 설정으로 ID/PW↔소셜 연동 지원 (실테스트 확인)
- **서비스 전환 용이** — 경쟁사에서 이전 시 기존 @s 회원 데이터 유지 (이메일 기반 매칭)
- **수파레인 생태계** — supa24(Discord 커뮤니티 커머스), 오팀장(AI 쇼핑몰 운영), supaMov(AI 모델 이미지), VuDrop(포장 검수) 등 연동 가능한 자체 서비스 보유
- **크레딧 쿠폰 시스템** — 번개가입(무료) 가입 → 생태계 서비스 크레딧 쿠폰 발급 → 크로스셀링

### Weaknesses (약점)
- **브랜드 인지도 없음** — 신규 서비스, 설치 수 0
- **마케팅 부가 기능 없음** — 미니 배너, CRM, 알림톡 등 미제공
- **카카오 채널 연동 없음** — 경쟁사 전체가 제공하는 필수 기능
- **관리형 서비스 미제공** — 설치 지원/운영 대행 인력 없음

### Opportunities (기회)
- **ScriptTag 활용 확장** — 이미 전 페이지에 스크립트 삽입 → 미니 배너, 팝업, 온사이트 마케팅 구현 가능
- **글로벌 쇼핑몰 타겟** — 해외 고객 대상 쇼핑몰에서 경쟁사가 커버하지 못하는 영역
- **무료/저가 전략** — 알파푸시의 무료 모델이 시장에서 먹히고 있음
- **경쟁사 약점 공략** — 해외 소셜 미지원 + 높은 가격
- **MZ 타겟 Discord 커뮤니티 커머스** — 번개가입 Discord 로그인 + supa24 연동 → 경쟁사 전체 불가능한 고유 포지션
- **VuDrop 포장 영상 알림** — ScriptTag으로 주문 포장 동영상 등록 시 고객 알림/링크 공유 → 경쟁사 전무한 기능

### Threats (위협)
- **진입장벽 낮음** — 우리 서비스에서 다른 서비스로의 이전도 쉬움
- **알파푸시 무료 모델** — 무료 + 올인원으로 빠르게 시장 확대 중
- **1초가입 시장 지배** — 13,856+ 설치로 사실상 표준
- **카카오 싱크 의존도** — 한국 시장에서 카카오 싱크가 사실상 필수

  - 카카오 채널 연동 ON → 과금
  - supa24 커뮤니티 연동 ON → 과금
  - VuDrop 포장 알림 ON → 과금
- **크레딧 쿠폰**: 가입 자체로 수파레인 생태계 서비스 크레딧 제공 → 자연스러운 업셀
- **이전 비용 최소화**: 언제든 해지 가능, 기존 회원 데이터 유지 → 도입 결정 쉽게
