## 3. 기존 회원-소셜 계정 연결

> **[2026-03-31 업데이트] 카페24 기본 기능으로 해결됨 — 별도 구현 불필요**

### 해결 방법 (실테스트 확인)

카페24 "회원가입 시 SNS 계정 연동" 설정을 활성화하면 별도 구현 없이 기존 ID/PW 회원과 소셜 계정 연동이 가능하다.

**플로우 (카페24 기본 제공):**
1. 카페24 관리자 → 회원 설정 → "회원가입 시 SNS 계정 연동" ON
2. 기존 회원과 동일 이메일로 SSO 로그인 시 → 카페24가 연동 안내 화면 표시
3. 고객이 기존 계정 선택 → 연동 완료
4. 이후 소셜 로그인으로 기존 회원 계정에 바로 접근 가능

**번개가입의 역할:** 쇼핑몰 운영자에게 "회원가입 시 SNS 계정 연동 설정 활성화" 안내 가이드 제공만으로 충분.

**프로바이더 제한:**
- ✅ Google, Kakao, Naver, Facebook, Discord, LINE — 이메일 제공 → 연동 가능
- ⚠️ Apple — Hide My Email 기본 선택 → 연동 불안정
- ❌ X, Telegram — 이메일 미제공 → 연동 불가

## 7. 모바일 환경 소셜 로그인 테스트

> 모바일 사용자가 대다수이므로 각 소셜 로그인별 모바일 환경 테스트가 필수.
> **우선순위: 서비스 개시 전 필수 작업**

### 테스트 항목

1. **팝업 ↔ 기존 페이지 전환** — 모바일 브라우저에서 팝업이 정상 동작하는지, 원래 페이지로 자연스럽게 돌아오는지
2. **프로바이더별 모바일 테스트**
   - Google: 모바일 OAuth 팝업/리다이렉트
   - Kakao: 카카오톡 앱 연동 (앱투앱)
   - Naver: 네이버 앱 연동
   - Apple: Safari/iOS 전용 플로우
   - Discord: 모바일 웹 OAuth
   - Facebook: Facebook 앱 연동
   - LINE: LINE 앱 연동
   - Telegram: 모바일 위젯 로그인
   - X: 모바일 웹 OAuth
3. **카페24 모바일 스킨 호환성** — 모바일 스킨에서 위젯 레이아웃, 버튼 크기, 터치 영역
4. **"회원가입 시 SNS 계정 연동" 화면** — 모바일에서 연동 선택 화면 UX 확인
5. **iOS Safari / Android Chrome** — 각 브라우저별 팝업 차단 정책 대응
