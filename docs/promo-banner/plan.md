# 번개가입 카페24 홍보 배너(1852×840) 제작 계획

**작성일**: 2026-04-20
**대상 규격**: 카페24 앱스토어 **키비주얼 슬라이드 배너** (메인 노출용)
**참조 가이드**: `docs/cafe24_banner_production_guide.md` (VuDrop 사례)
**관련 자산**: 앱 소개 배너 `docs/banner-v8*-2x.png`(740×416), 공식 아이콘 `docs/icon.png`

## 1. 배너 개요

| 항목 | 값 |
|---|---|
| 제작 사이즈 | **1852 × 840** (실사이즈 2배) |
| 실제 노출 사이즈 | 926 × 420 |
| 포맷 | PNG (투명 배경 불필요, 단색/그라디언트 기반) |
| 좌우 세이프존 | 각 **65px** |
| 하단 세이프존 | y=720~840 (페이지 인디케이터, 재생 버튼) |
| 카피 입력 영역 | 좌측 상단 y=0~300, x=130~ (**비워두기**) |
| 배경 조건 | **어두운 톤 필수** (흰색 텍스트 합성) |

> **규칙**: 메인/서브 카피는 이미지에 포함시키지 않는다. 카페24 서버가 흰색으로 실시간 합성. 좌측 상단을 반드시 어두운 네이비로 유지.

## 2. 카피 (별도 납품)

배너 740×416(v7/v8)에서 합의된 카피 톤을 그대로 계승. 메인 1개 확정 + 서브 1개 확정, 예비안 2개.

### 메인 카피 후보 (공식 가이드 권장: 30자 이내)

| 안 | 카피 | 글자수 | 특징 |
|---|---|---|---|
| **★ A (확정 제안)** | **회원가입이 번개처럼 끝납니다** | 14 | 브랜드명(번개가입) 자체 설명, 이름-기능 일치 |
| B | 번개처럼 가입, 소셜 1-클릭 | 14 | 행동 유도형, 광고 톤 |
| C | 소셜 계정 하나면, 회원가입 끝 | 15 | 고객 혜택 직설 |

### 서브 카피 후보 (공식 가이드 권장: 50자 이내)

| 안 | 카피 | 글자수 | 특징 |
|---|---|---|---|
| **★ 1 (확정 제안)** | **카카오·네이버·구글 1-클릭 가입 · 관리자에서 디자인 바로 수정** | 32 | 고객 가치 + 운영자 가치 동시 전달 |
| 2 | 복잡한 가입 폼 없이 소셜 계정 하나로 · 월 0원으로 시작 | 27 | 가격 임팩트 앞세움 |
| 3 | 소셜 6종 지원 · 설치 1분 · 관리자 슬라이더로 디자인 커스텀 | 33 | 스펙 나열형 |

### 납품 사양
```
[파일]    suparain-key-visual-1852x840.png
[메인]    회원가입이 번개처럼 끝납니다
[서브]    카카오·네이버·구글 1-클릭 가입 · 관리자에서 디자인 바로 수정
```

## 3. 우측 비주얼 컨셉 3안

좌측 상단 y=0~300은 카피 합성 영역이므로 비워둠. **메인 비주얼은 우측 40~50%에 단일 포커스**.

### 컨셉 1 ★ — **번개 심볼 + 소셜 로고 오빗 링** (추천)

```
┌──────────────────────────────────────────────┐
│                                              │
│  [카피 합성 영역]         ⚡ (거대 번개)    │
│                          ╭─ G ─╮            │
│                         K      N            │
│                          ╲─Apl Dc Tg─╯      │
│                                              │
│  [브랜드 칩]                                │
└──────────────────────────────────────────────┘
```

- 우측 중앙에 번개 심볼(높이 ~560px)을 히어로 요소로 배치
- 소셜 6종(G, K, N, Apple, Discord, Telegram) 아이콘을 번개 주변 원형/호 형태로 배치
- 번개에서 소셜 로고로 뻗는 미세 방사형 빛 → "번개가입이 모든 소셜을 하나로"
- **강점**: 브랜드 시그니처(번개) 최대 노출 + 6개 프로바이더 즉각 전달
- **약점**: 전형적, 독창성 다소 부족

### 컨셉 2 — 카페24 로그인 목업 위 "번개가입" 버튼 glow

```
┌──────────────────────────────────────────────┐
│                                              │
│  [카피 합성 영역]      ┌─ 로그인 박스 ─┐   │
│                        │  ID     │        │   │
│                        │  PW     │        │   │
│                        │ ╭─⚡번개가입─╮  │   │
│                        │ │ 눈부신 glow│ ←→ 1초 │
│                        │ ╰──────────╯ │   │
│                        └───────────────┘   │
│  [브랜드 칩]                                │
└──────────────────────────────────────────────┘
```

- 실제 카페24 로그인 박스 목업 (반투명 유리 효과)
- 박스 내부에 "번개가입" 버튼만 강하게 glow — "이 버튼 하나로 끝"
- 버튼 우측에 "1초" 라벨
- **강점**: 실제 사용 컨텍스트를 보여줌, 설치 후 어떻게 보일지 한눈에
- **약점**: 목업 디테일이 많아 축소 시 뭉개질 수 있음

### 컨셉 3 — 거대 "1초" 숫자를 번개가 관통

```
┌──────────────────────────────────────────────┐
│                                              │
│  [카피 합성 영역]         ╭─ 1 ─╮ ⚡       │
│                           │ (거대 숫자)│    │
│                           │   초      │    │
│                           ╰───────────╯    │
│                          G K N Apl Dc Tg   │
│  [브랜드 칩]                                │
└──────────────────────────────────────────────┘
```

- 우측에 "1초" 초대형 숫자 (높이 ~480px, 900 weight)
- 번개가 숫자를 사선으로 관통
- 하단에 소셜 로고 6종 작게 나열
- **강점**: 가장 임팩트 강함, "1초"라는 핵심 가치 즉각 전달
- **약점**: "1초가입"이라는 경쟁 서비스와 키워드 충돌 가능

### 권장: **컨셉 1**
- 배너 740×416(v8)의 시각 언어와 가장 일관
- 번개 심볼(아이콘 v2-a)을 재활용 → 브랜드 자산 통합
- 경쟁사와 차별점(6개 프로바이더 다양성)을 직관적 노출

## 4. 브랜드 칩 (VuDrop 관행 따라 2버전 제작)

### 구성 요소
- 번개 심볼 미니(48×48, 아이콘 v2-a 축소)
- "번개가입" 워드마크 (굵은 산세리프, 한글)
- 보조 라벨: "· 카페24 공식 앱"

### 위치 옵션
| ID | 위치 | 비고 |
|---|---|---|
| **A (기본)** | 하단 좌측 full size (높이 72px) | 안정적, 카피 합성과 간섭 없음 |
| **B (대안)** | 상단 좌측 compact (높이 48px) | 카피 영역과 공존, 합성 결과 확인 후 폴백 |

## 5. 배경 규칙

### 컬러 팔레트 (배너 v8/아이콘 v2-a와 통일)
- 주 배경: `#1E3A8A → #142759 → #0B1638` (radial gradient)
- 좌측 카피 영역: **반드시 가장 어두운 톤(#0B1638)** 으로 유지
- 포인트 컬러: 오렌지 `#FF8800 → #FFB347`

### 배경 장식 (축소 시 뭉개지지 않는 수준으로)
- 우측 상단에 블러 블루 글로우 서클 (r=260px, opacity 0.22)
- 좌측 하단에 미세 오렌지 글로우 (r=200px, opacity 0.10)
- 전체 미세 그리드 (rgba 0.025, 80px 간격)

## 6. 검증된 수치 vs 금지 수치

### ✅ 배너에 사용 가능 (공식 스펙/구현 기반)

| 항목 | 값 | 출처 |
|---|---|---|
| 소셜 프로바이더 | **6종** (Google / Kakao / Naver / Apple / Discord / Telegram) | 구현 완료 — `settings.tsx` |
| Plus 월간 요금 | **월 ₩6,900** (부가세 별도) | `billing.tsx:70` |
| Plus 연간 요금 | **연 ₩79,000** | `billing.tsx:71` |
| 무료 플랜 | **월 0원** | 공식 가격표 |
| 설치 소요 시간 | **약 1분** | 설치 과정 실측 |
| 디자인 슬라이더 | **6종** (너비/높이/간격/라운드/아이콘 간격/여백) | `settings.tsx:644-679` |
| 디자인 프리셋 | **5종** (컬러/모노톤/테두리/호버/아이콘만) | `settings.tsx` |
| 타사 경쟁 앱 최저가 | **월 ₩49,900** | 1초가입 Basic(외부 공개가) |

### ❌ 배너에 쓰면 안 되는 수치 (근거 없음)

- "가입률 N% 상승"
- "구매 전환 N% 증가"
- "이탈률 N% 감소"
- "평균 가입 시간 N초 단축"

> 이런 성과 수치는 **실측 데이터가 쌓이기 전까지 금지**. 카피·비주얼·뱃지 어디에도 사용 X.

## 10. 이 문서에서 확정할 3가지

1. **메인 카피 A/B/C 중 무엇**? (기본값: A "회원가입이 번개처럼 끝납니다")
2. **서브 카피 1/2/3 중 무엇**? (기본값: 1 "카카오·네이버·구글 1-클릭 가입 · 관리자에서 디자인 바로 수정")
3. **비주얼 컨셉 1/2/3 중 무엇**? (기본값: 1 "번개 심볼 + 소셜 오빗 링")

확정되면 `drafts/concept-{n}.html` 작성 → Chrome Headless 렌더 → 축소 가독성 확인 → 최종 납품본 생성 순서로 진행.
