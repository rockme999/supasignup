## ScriptTag란?

- **정의**: 카페24 자동 스크립트 삽입 API
- **역할**: 번개가입 설치 시 쇼핑몰 모든 페이지에 자동으로 로그인 버튼 + 부가기능 렌더링
- **효과**: 추가 설치 없이 모든 기능 즉시 활성화 가능

```
설치 → 카페24 자동 삽입 → 쇼핑몰 전 페이지에 <script> 태그 추가
→ 로그인 버튼 + 배너 + 팝업 등 모두 자동 동작
```

## ScriptTag 기반 부가기능 로드맵

### Phase 1 (MVP — 필수)
- ✅ **소셜 로그인 9종** (거의 완료)
- ✅ **스마트 버튼** (이전 로그인 기억)
- ⚠️ **미니 배너** (1초가입 스타일 슬라이드 카드 4종 + AI 카피라이터)
- ⚠️ **카카오 채널 연동** (가입 시 자동 친구 추가)

### Phase 2 (출시 후 1~2개월)
- **소셜 프루프 위젯** ("방금 N명이 가입했어요" 실시간 알림)
- **이탈 팝업** (마우스 이탈 감지 시 팝업)
- **프로바이더별 분석 대시보드** (전환율, 일별 추이 등)

### Phase 3 (차별화 — AI)
- **AI 스마트 배너** (자동 카피/이미지 생성)
- **AI A/B 테스트** (버튼 순서, 문구 자동 최적화)
- **AI 세그먼트 분석** (프로바이더별 구매 전환율)
- **Discord 커뮤니티 연동** (Discord 로그인 시 서버 자동 초대 — MZ 킬러)

### Phase 4 (생태계)
- **크레딧 쿠폰** (가입 시 생태계 서비스 크레딧 자동 발급)
- **VuDrop 포장 알림** (주문 포장 영상 등록 시 고객 알림)

## 기술 스펙 핵심

### 위젯 구조 (Vanilla JS)
```
/widget/buttons.js (ScriptTag로 삽입)
  → 로그인/회원가입 페이지 감지
  → /api/widget/config 호출 (활성 프로바이더 조회)
  → localStorage에서 이전 로그인 provider 확인
  → 버튼 HTML 렌더링 (이전 provider 강조)
```

### 부가기능 확장 방식
```
모든 부가기능 = ScriptTag + KV 캐시 + D1 데이터

예) 미니 배너
  1. 운영자가 대시보드에서 배너 생성
  2. 설정이 KV에 캐시됨
  3. 쇼핑몰 모든 페이지의 ScriptTag가 KV 설정 읽음
  4. 해당 배너를 렌더링
  → 추가 설치 필요 없음 (ON/OFF만으로 활성화)
```

### KV 캐시 설계
- `widget_config:{shop_id}`: 위젯 설정 (5분 TTL)
- `oauth_session:{state}`: OAuth 세션 (10분 TTL)
- `access_token:{token}`: 로그인 토큰 (2시간 TTL)
- `dashboard_session:{token}`: 대시보드 세션 (24시간 TTL)

## 1초가입 vs 번개가입

### 1초가입의 강점
- 로그인 페이지 **완전 대체** (특허 UI 설계)
- 하단 혜택 배너 (슬라이드 카드 4종)
- CRM 자동화 (KeepGrow 플랫폼)
- 시장 압도 (13,856+ 설치)

### 번개가입의 대응
- 미니 배너는 Phase 1에서 필수 구현 ✅
- 로그인 페이지 완전 대체는 기술적 제약 (카페24 구조)
- CRM은 데이터 축적 후 Phase 3에서 추진
- **9종 소셜 + AI = 경쟁사 없는 차별점** ✨

---

## 알파푸시 vs 번개가입

### 알파푸시의 강점
- **무료 모델** (6개월 만에 6,000개 쇼핑몰)
- 회원가입 + CRM + 마케팅 + 분석 수직 통합

### 번개가입의 우위
- 9종 소셜 + 무료 = 알파푸시보다 우수
- **생태계 연동** (경쟁사 불가능)
  - supa24: Discord 커뮤니티 커머스
  - VuDrop: 포장 영상 알림
  - 오팀장: AI 운영 연동
- **AI 배너 카피라이터** (경쟁사 미사용)

## 수익 모델

### 무료 (영구)
- 소셜 로그인 9종
- 스마트 버튼
- 마이페이지 연동 위젯
- 기본 통계

### 유료 (부가기능 ON/OFF)
- 미니 배너 복수: 월정액
- 카카오 채널 연동: 월정액
- 소셜 프루프 위젯: 월정액
- 이탈 팝업: 월정액
- AI 배너/A/B 테스트: 월정액
- Discord 커뮤니티 연동: 월정액

### 이탈 비용 자동 증가 (Lock-in)
```
설치 직후: 이탈 비용 낮음 (데이터 없음)
  ↓
3개월 후: 중간 (분석 데이터 축적)
  ↓
6개월 후: 높음 (A/B 테스트 최적화 결과)
  ↓
1년 후: 매우 높음 (대체 불가능한 데이터 자산)
```

## MVP 구현 일정 (7주)

| 주차 | 작업 |
|------|------|
| 1주 | 프로젝트 셋업 + cafe24-client 포팅 |
| 2주 | OAuth Provider 핵심 구현 |
| 3주 | 소셜 OAuth 연동 (Google, Kakao, Naver, Apple) |
| 4주 | **위젯 + 스마트 버튼** (ScriptTag 핵심) |
| 5주 | 대시보드 (쇼핑몰 관리 + 프로바이더 선택) |
| 6주 | 대시보드 (통계 + 과금) + E2E 테스트 |
| 7주 | 버그 수정 + 최적화 + 앱스토어 준비 |
